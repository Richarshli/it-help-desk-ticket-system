# IT Help Desk Ticket System

A beginner-friendly web application for creating and tracking IT help desk support tickets.

## Purpose

This project demonstrates a simple IT support workflow for documenting technical issues, tracking ticket status, assigning priority, and recording resolution notes.

## Features

- Create IT support tickets
- Track user name
- Track issue type
- Assign priority level
- Track ticket status
- Add troubleshooting and resolution notes
- View tickets in a table
- Beginner-friendly HTML, CSS, and JavaScript project

## Files

- `index.html` - Main web page
- `style.css` - Styling and layout
- `script.js` - Ticket tracker functionality

## Example Use Case

An IT support representative can enter a user issue, assign a priority, track the status, and document troubleshooting steps or resolution notes.

## Important Note

This project is for educational and portfolio purposes only. Do not enter real customer, employee, or company data.
